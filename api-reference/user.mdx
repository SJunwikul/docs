---
title: "GET /v1/user"
description: "Retrieve authenticated user's profile and credits"
---

## Endpoint

```
GET https://oauth.joinares.com/v1/user
```

## Authentication

Requires a valid OAuth 2.0 Bearer token in the Authorization header.

```http
Authorization: Bearer YOUR_OAUTH_ACCESS_TOKEN
```

## Request

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | OAuth 2.0 Bearer token |

### Query Parameters

None

### Request Body

None

## Response

### Success Response (200 OK)

```json
{
  "id": "123e4567-e89b-12d3-a456-426614174000",
  "email": "user@example.com",
  "name": "John Doe",
  "credits": 150,
  "picture": "https://example.com/avatar.jpg"
}
```

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique user identifier (UUID) |
| `email` | string | User's email address |
| `name` | string | User's display name |
| `credits` | number | Current credit balance |
| `picture` | string (optional) | User's profile picture URL |

## Error Responses

### 401 Unauthorized

Invalid or missing OAuth token.

```json
{
  "error": "Unauthorized",
  "message": "Invalid or missing authentication token"
}
```

### 403 Forbidden

Token has expired.

```json
{
  "error": "Forbidden",
  "message": "Token has expired"
}
```

### 500 Internal Server Error

Server error occurred.

```json
{
  "error": "Internal Server Error",
  "message": "An unexpected error occurred"
}
```

## Code Examples

<CodeGroup>

```typescript TypeScript SDK
import { Ares } from 'ares-sdk';

const client = new Ares({
  accessToken: 'your_oauth_access_token',
});

const user = await client.user.retrieve();

console.log(user.id);      // "123e4567-e89b-12d3-a456-426614174000"
console.log(user.email);   // "user@example.com"
console.log(user.name);    // "John Doe"
console.log(user.credits); // 150
```

```javascript cURL
curl https://oauth.joinares.com/v1/user \
  -H "Authorization: Bearer YOUR_OAUTH_ACCESS_TOKEN"
```

```javascript Fetch API
const response = await fetch('https://oauth.joinares.com/v1/user', {
  headers: {
    'Authorization': 'Bearer YOUR_OAUTH_ACCESS_TOKEN',
  },
});

const user = await response.json();
```

</CodeGroup>

## Use Cases

### Check Credit Balance

Before providing a service, check if the user has sufficient credits:

```typescript
const user = await client.user.retrieve();

if (user.credits < requiredCredits) {
  throw new Error('Insufficient credits');
}
```

### Display User Profile

Show user information in your application:

```typescript
const user = await client.user.retrieve();

return (
  <div>
    <img src={user.picture} alt={user.name} />
    <h2>{user.name}</h2>
    <p>{user.email}</p>
    <p>Credits: {user.credits}</p>
  </div>
);
```

### Conditional Features

Enable/disable features based on credit balance:

```typescript
const user = await client.user.retrieve();

const features = {
  basicFeature: user.credits >= 10,
  premiumFeature: user.credits >= 50,
  enterpriseFeature: user.credits >= 200,
};
```

## Rate Limiting

This endpoint is rate-limited to prevent abuse. Exceeding the rate limit will result in a 429 error.

<Warning>
  Cache user data when appropriate to avoid unnecessary API calls. Credits may change between calls.
</Warning>
